# AI-Smart-Surveillance-System

AI-powered smart surveillance system featuring real-time video monitoring, face recognition, recording management, and a web dashboard.

## Features

- Real-time video monitoring through a web dashboard.
- Face detection and recognition using AI-based embeddings.
- Face registration for new users.
- Video recording functionality.
- Recording history with play and download options.
- Admin user management dashboard.
- Role-based access for admin and employee users.

## Technologies Used

- Python
- Flask
- OpenCV
- MTCNN
- FaceNet
- HTML
- CSS
- JavaScript
- CSV-based local storage

## Results

- Implemented 5 main modules: Login, Live Dashboard, Face Recognition, User Management, and Recording History.
- Face recognition result displays user name, ID, confidence percentage, and detection time.
- Video recording supports live recording and recording history management.
- Recording history dashboard displays total recordings, file size, date, and available actions.
